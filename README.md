# 🥠 Fortune Cookie로 보는 2026년도 운세 

> 당신의 2026년 운세를 재미있게 확인할 수 있는 인터랙티브 포춘쿠키 웹 애플리케이션입니다.
> 클릭 한 번으로 쿠키가 깨지고, 다양한 파티클과 쪽지, 운세 메시지가 나타납니다! 🎉

---

## 소개

* **닫힌 쿠키**를 클릭하면 쿠키가 깨지면서 **파티클 효과**가 발생
* **쪽지와 운세 메시지**가 자연스럽게 나타남
* **운세 유형별 스타일**(금전, 사랑, 건강, 행운, 성공)을 구분하여 시각적 효과 제공
* **다시 뽑기 버튼**으로 쿠키를 리셋하고 반복 플레이 가능
* **모바일과 데스크탑 모두 대응**, 클릭과 터치 모두 지원

---

## 주요 기능 및 특징

### 1. 인터랙티브 포춘쿠키

* 클릭 전: 귀여운 닫힌 쿠키 이미지
* 클릭 후: 쿠키가 좌우로 깨지며 쪽지가 튀어나오는 애니메이션

### 2. 파티클 효과

* 12개의 파티클이 폭발하듯 퍼지며 화려한 시각 효과 제공
* 파티클 색상: 금색, 주황색, 빨강 등
* 애니메이션 지속 시간: 1초, 자연스럽게 사라짐

### 3. 운세 쪽지

* 쪽지 디자인: 흰색 바탕 + 운세 유형별 테두리와 그림자
* 운세 유형:

  * 금전(`fortune-money`)
  * 사랑(`fortune-love`)
  * 건강(`fortune-health`)
  * 행운(`fortune-luck`)
  * 성공(`fortune-success`)
* 각 유형에 맞는 색감과 그림자, 배경 그라데이션 적용

### 4. 재도전 기능

* “다시 뽑아보기” 버튼으로 쿠키와 파티클을 초기화
* 여러 번 반복해서 운세 확인 가능

## 코드 핵심 포인트

1. **쿠키 클릭 이벤트**

```jsx
const handleClick = () => {
  if (isBroken) return;
  setIsBroken(true);       // 쿠키 깨진 상태
  setShowParticles(true);  // 파티클 애니메이션 실행
  onOpen();                // 부모 컴포넌트에서 운세 선택
};
```

2. **쪽지에 운세 텍스트 표시**

```jsx
<div className={`paper ${getFortuneClass()}`}>
  {fortune.icon && <div className="fortune-icon">{fortune.icon}</div>}
  <p>{fortune.text}</p>
</div>
```

3. **CSS 애니메이션**

* 쿠키 흔들림: `@keyframes wobble`
* 깨진 조각 이동: `@keyframes breakLeft`, `breakRight`
* 파티클 폭발: `@keyframes particleExplode`
* 쪽지 등장: `@keyframes paperUp`

## 사용 방법

1. 페이지 접속 후 쿠키 클릭 → 쿠키 깨짐 + 파티클 폭발 + 운세 쪽지 등장
2. 원하는 운세 메시지 확인
3. “다시 뽑아보기” 클릭 → 쿠키와 파티클 초기화, 반복 가능

## 운세 메시지 (Fortunes)

Fortune Cookie 앱은 클릭할 때마다 **랜덤으로 운세 메시지**를 보여줍니다.
모든 운세 메시지는 `fortunes.js`에서 관리하며, 다양한 상황과 재미있는 상상을 담고 있습니다.

### 예시 메시지

* 2026년, 갑자기 배달 음식이 무료로 올 확률이 높습니다. 준비하세요!
* 뜻밖의 이메일 하나가 당신의 운명을 바꿀 수 있어요. 지금 당장 이메일함을 열람해보세요!
* 친구가 빌린 돈을 깜빡하고 갚을지도?! 웃으며 받아주세요.
* 갑자기 내린 비 속에서 발견한 1,000원의 행운이 당신을 기다리고 있습니다.
* 올해는 꿈속에서 만난 고양이가 중요한 힌트를 줄지도 몰라요.
* 갑작스러운 여행 계획이 최고의 추억으로 이어집니다. 같이 여행 갈 사람~
* 오늘의 점심 메뉴가 당신에게 영감을 줄 수도?!
* 올해는 우산을 꼬박꼬박 챙겨볼까요? 누군가와의 우연한 만남이 생길지도?!
* 소소한 실수가 대박 행운으로 이어질 수 있습니다. 실수해도 괜차너 ~~
* 하늘을 바라보며 한숨을 쉬면, 바로 옆에 행운이 숨어있을지도?!

> 전체 운세 메시지는 `src/fortunes.js`에서 확인 가능하며, 자유롭게 추가하거나 수정할 수 있습니다.
